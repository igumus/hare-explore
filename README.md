# Hare Breakout Experiment 🎮

A small game experiment written in **Hare**.

This project is my **first attempt at game programming**. The main goal is to explore **basic game mechanics** while learning and experimenting with the **Hare programming language**.

## About the Project

This project is inspired by classic **Breakout-style game play**. It serves as a playground to experiment with simple game logic, mechanics, and structure using Hare.

The focus is not on building a full production-ready game, but on learning how game systems can be implemented.

## Current Features

* Classic **Breakout-style mechanics**
* Targets (blocks) with **different rigidity levels**
* Basic collision and destruction logic

## Planned Features

* **Bonus mechanics** triggered by hitting special targets
* Additional gameplay experiments and mechanics
* More target behaviors and variations
* General gameplay improvements

## Goals

* Learn fundamental **game development concepts**
* Experiment with **game loops, collisions, and mechanics**
* Improve familiarity with the **Hare language**
* Build a small but fun experimental project

## Status

🚧 **Work in progress**

This project is experimental and features may change as I continue learning and exploring new ideas.
